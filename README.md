# üè¶ Fixed Deposit Subscription Prediction ‚Äì Banking Call Center Domain

This project analyzes customer interaction data from a bank's call center to predict whether a client will subscribe to a **fixed deposit**. The goal is to help banks **optimize marketing strategies** by targeting the right customers using data-driven machine learning models.

## üéØ Problem Statement

In the competitive banking sector, understanding which customers are most likely to subscribe to financial products is critical. This project uses call center data to build a machine learning model that predicts **fixed deposit subscription**, helping to improve conversion rates and customer targeting.

---

## üìä Data Overview

The dataset includes:
- Customer demographics (age, job, marital status, education)
- Call and contact details (contact type, duration, last contact day/month)
- Campaign information (number of contacts, previous outcome)
- **Target variable**: `y` (Yes/No - whether the customer subscribed)

---

## üîç Key Insights

- Customers with longer call durations are more likely to subscribe.
- Retired and student segments show higher subscription rates.
- A successful previous campaign increases current subscription likelihood.
